Μοίρασμα αρχείων – Δημόσια αρχεία

Τα δικαιώματα πρόσβασης ανατίθενται σε χρήστες ή ομάδες χρηστών ανά αρχείο ή κατάλογο, προσδιορίζοντας έτσι ποιός έχει πρόσβαση σε ποιό αρχείο και τι είδους πρόσβαση.

Δικαιώματα πρόσβασης

Το σύστημα χρησιμοποιεί τα εξής δικαιώματα πρόσβασης:

  • Read: επιτρέπει στο χρήστη να βλέπει το αρχείο ή τον κατάλογο και τα μετα-δεδομένα του.

  • Write: επιτρέπει στον χρήστη να τροποποιήσει το αρχείο ή τον κατάλογο και τα μετα-δεδομένα του.

  • Modify Access: επιτρέπει στον χρήστη να τροποποιήσει τα δικαιώματα πρόσβασης του αρχείου ή καταλόγου.

Ανάθεση δικαιωμάτων πρόσβασης

Η ανάθεση των δικαιωμάτων πρόσβασης γίνεται μέσω του διαλόγου Properties, όπως περιγράφεται εδώ για καταλόγους και εδώ για αρχεία. Η παρακάτω εικόνες δείχνουν παραδείγματα ορισμού δικαιωμάτων για κατάλογο και αρχείο.

Ισχύουν οι εξής συμβάσεις-κανόνες:

  • Ο ιδιοκτήτης των αρχείων αυτόματα λαμβάνει όλα τα δικαιώματα στους καταλόγους και τα αρχεία του (σημειώνεται ως owner στο διάλογο)

  • Ο ορισμός δικαιωμάτων γίνεται ανά χρήστη ή ομάδα χρηστών. Το σύστημα επιτρέπει σε κάθε χρήστη να ορίζει ομάδες χρηστών για χρήση στην ανάθεση δικαιωμάτων πρόσβασης στα αρχεία του. (εδώ αναφέρεται ο τρόπος ορισμού ομάδων).

  • Η ανάθεση δικαιωμάτων σε κατάλογο αντιγράφεται αυτόματα σε όλα τα περιεχόμενα του. Στη συνέχεια ο χρήστης μπορεί να διοφοροποιήσει τα δικαιώματα ανά αρχείο του, μόλις όμως τροποιήσει τα δικαιώματα στον κατάλογο, τότε και πάλι αυτά θα συγχρονιστούν με τον κατάλογο.

  • Όπως φαίνεται στον διάλογο του αρχείου (βλέπε εικόνα παραπάνω), ο χρήστης μπορεί να κάνει ένα αρχείο "δημόσιο", δηλαδή να επιτρέψει ανάγνωση και πρόσβαση σε αυτό χωρίς να απαιτείται έλεγχος πρόσβασης. (επιλογή "Make Public"). Σε αυτή την περίπτωση, το αρχείο είναι διαθέσιμο μέσω του URI που εμφανίζεται στο κάτω μέρος του διαλόγου

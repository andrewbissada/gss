Πρόσβαση μέσω webDAV

Το σύστημα παρέχει πρόσβαση στα αρχεία και τους καταλόγους του χρήστη και μέσω του πρωτοκόλλου webDAV, ή τα λεγόμενα web folders. Το webDAV υποστηρίζεται από Windows 2000, XP (όχι Vista), διάφορες εκδόσεις UNIX/Linux και Mac OS X. Η δομή του δεν επιτρέπει authentication μέσω Shibboleth καθώς απαιτεί username/password authentication μέσα από το ίδιο το πρωτόκολλο. Έτσι, όταν ο χρήστης ορίζει το WebDAV folder θα πρέπει να δώσει τα εξής:

  • URL: http://pithos.grnet.gr/webdav

  • username: το username που δίνει το σύστημα μέσω του μενού Settigns -> User Credentials (εικόνα παρακάτω)

  • password: το token που δίνει το σύστημα μέσω του μενού Settigns -> User Credentials (εικόνα παρακάτω)


Στοιχεία πρόσβασης για WebDAV access

Προσοχή: το token δημιουργείται κάθε φορά που κάνει login ο χρήστης μέσω Shibboleth και καθώς ισχύει μόνο για 24 ώρες, θα πρέπει να έχει προηγηθεί login εντός αυτού του ορίου.


WebDAV (Web Folders) στα MS Windows

Η υποστήριξη του WebDAV (τα λεγόμενα Web Folders) στα Windows XP έχει σοβαρά προβλήματα χωρίς το SP3. Το fix Software Update for Web Folders (KB907306) επίσης λύνει κάποια προβλήματα.

Κάποια σημεία που πρέπει να έχουν υπόψη οι χρήστες του Pithos WebDAV client σε περιβάλλον Windows XP:

  • Η σύνδεση γίνεται ως εξής:
    • Ανοίγουμε τον Internet Explorer (IE6 ή ΙΕ7)

    • Ctrl-o και ανοίγει ο διάλογος Οpen

    • Στο input field δίνουμε το webdav URL: http://pithos.grnet.gr/webdav

    • Επιλέγουμε (check) την επιλογή "Open as Web Folder" και πατάμε "ΟΚ"

    • Ανοίγει παράθυρο Explorer όπου εισάγουμε το username και token όπως περιγράφηκε παραπάνω

    • Αν ο έλεγχος πρόσβασης είναι επιτυχής, γίνεται η σύνδεση και δημιουργείται αυτόματα και το κατάλληλο network place "webdav on pithos.grnet.gr"

    • Κάθε επόμενη σύνδεση μπορεί να γίνει από το My Network Places -> webdav on pithos.grnet.gr και να δοθεί το κατάλληλο password / token (αν έχει λήξει)


  • Τα Windows Web Folders εμφανίζονται ως network places και όχι local folders ή SMB shares. Αυτό σημαίνει ότι οι εφαρμογές με τις οποίες "ανοίγουμε" αρχεία πρέπει να υποστηρίζουν πρόσβαση / διαχείριση αρχείων σε network places ώστε να μπορούμε να δουλέψουμε με αυτά ανοίγοντάς τα απευθείας από το "webdav on pithos.grnet.gr" (π.χ. με double-click ή File->Open->Network places ...). Για παράδειγμα οι εφαρμογές του Open Office και του MS Office υποστηρίζουν τέτοια λειτουργία, αλλά απαιτούν στην πρώτη πρόσβαση να δοθεί username / password, έτσι με τα σχετικά αρχεία μπορεί κανείς να δουλέψει απευθείας στο web folder και όχι σε τοπικό αντίγραφο. Αντίθετα, ο Acrobat Reader δεν λειτουργεί σωστά με web folders. Σε κάθε περίπτωση λειτουργούν οι λειτουργίες copy/cut/paste και drag and drop για την ανταλλαγή αρχείων μεταξύ του Pithos και των δίσκων και shares που βλέπουν τα Windows XP του χρήστη.

Η υποστήριξη του WebDAV στα Vista είναι πρακτικά ανύπαρκτη ακόμη και με την εγκατάσταση του KB907306. Παρότι το web folder εμφανίζεται, ο χρήστης δεν μπορεί να χειριστεί αρχεία, κλπ.
